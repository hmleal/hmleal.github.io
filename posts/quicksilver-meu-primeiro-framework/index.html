Quicksilver Meu Primeiro Framework

April 12, 2019 by Henrique Leal

Recentemente decidi entender mais como funcionam os frameworks para desenvolvimento web. Foi ai que então surgiu a ideia de criar um do zero.

O único requisíto seria não possuir nenhuma dependência externa. Ainda existem muitas coisas que podem ser implementadas mas já temos umas versão minimamente funcional
